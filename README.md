**[OnePRO](https://github.com/GaoSSR/OnePRO)** is my flagship open-source project: a stateless, algorithm-driven promotion engine built to solve complex discount stacking and optimal ordering problems in e-commerce pricing flows.

## Selected Technical Wins

- Fixed a GraalVM Native Image compatibility issue in Redisson by removing fragile reflection from the node conversion flow.
- Fixed an integer-overflow edge case in Redisson load balancers that could produce a negative index.
- Improved Maven dependency-resolution diagnostics for unresolved expressions, making failures easier to triage.
- Fixed a zero-length byte-buffer bug in Spring Boot's `InspectedContent` path.
- Landed multiple upstream code-quality fixes in Kafka and Netty across networking and test infrastructure.
