## About

I build production-oriented backend systems in the Java ecosystem, with a strong bias toward correctness, maintainability, and performance.

- Maintainer of [OnePRO](https://github.com/GaoSSR/OnePRO), a stateless promotion-computation engine for discount stacking and pricing optimization
- Verified merged upstream contributions in Redisson, Apache Maven, Apache Kafka, and Netty
- Active contributor to Spring Boot ecosystem code around buildpack and platform handling

> I do not just use open source infrastructure. I contribute back to it.
